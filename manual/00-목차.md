# OpenClaw 실전 매뉴얼 북

> AI 에이전트를 만들고, 팀으로 구성하고, 사업에 레버리지하는 완전 가이드
>
> by AI싱크클럽

---

## 이 매뉴얼의 대상

- OpenClaw로 AI 에이전트를 처음 만들어보려는 사람
- 이미 사용 중이지만 팀 시스템으로 확장하고 싶은 사람
- AI 에이전트로 사업 자동화/수익화를 원하는 사람

## 사전 준비물

- Mac (M1 이상 권장) 또는 Linux 서버
- 텔레그램 계정
- Anthropic API Key 또는 OpenAI API Key
- 터미널 기본 사용법

## 목차

### Phase 1: 에이전트의 집 구경

에이전트가 사는 곳을 이해하고, 첫 번째 에이전트를 세팅합니다.

| Ch | 제목 | 핵심 내용 |
|----|------|----------|
| 01 | 워크스페이스 — 에이전트가 사는 곳 | 디렉토리 구조, 핵심 파일 7개, 세션의 개념 |
| 02 | SOUL.md — 에이전트의 정체성 만들기 | Identity/Mission/Personality/Boundaries/Tone 5요소 |
| 03 | AGENTS.md — 업무 매뉴얼과 절대 규칙 | 행동 규칙, 실수→규칙 전환, 지침 vs 시스템 2중 방어 |
| 04 | 채널 연결 — 텔레그램/Slack 세팅 | 텔레그램 봇 생성, Slack 앱 설정, 채널 라우팅, 세션 분리 |

### Phase 2: 혼자 움직이게 하기

에이전트가 스스로 판단하고 일하게 만듭니다.

| Ch | 제목 | 핵심 내용 |
|----|------|----------|
| 05 | Heartbeat & Cron — 자동 실행의 기술 | HEARTBEAT.md 설계, cron job 분리, 시간 조건, 중복 방지 |
| 06 | 스킬(Skill) — 에이전트에게 능력 부여하기 | SKILL.md 구조, scripts/references/ 패턴, n8n 대체 |
